# social-media-app
Social Media App

Description

This is a simple Social Media web application developed using HTML, CSS, JavaScript, Node.js. The application allows users to register, log in, and access a social media home page where they can create posts.

Features

- User Registration
- User Login
- Home Page
- User Profile Section
- Create Posts
- Responsive User Interface

Technologies Used

- HTML
- CSS
- JavaScript
- Node.js


Project Structure

public/
│── register.html
│── login.html
│── home.html
│── style.css
│── script.js

server.js
package.json
README.md

How to Run

1. Install Node.js.
2. Install dependencies:
   npm install
3. Start the server:
   node server.js
4. Open your browser and visit:
   http://localhost:3000

Workflow

1. User Registration
2. User Login
3. Access Home Page
4. Create and View Posts
